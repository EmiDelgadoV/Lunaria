# 🌑 Lunaria - Rastreador de Fases Lunares

**Lunaria** es una aplicación de escritorio desarrollada en Python que permite calcular y visualizar el estado actual de la luna y predecir sus fases futuras con precisión astronómica.

## 🚀 Características Principales

* **Cálculo en Tiempo Real:** Utiliza la librería científica `ephem` para obtener datos astronómicos precisos.
* **Visualización Gráfica:** Dibuja la luna píxel a píxel usando `tkinter.Canvas`, representando sombras y porcentajes de iluminación.
* **Adaptabilidad Geográfica:** Hemisferio Sur** **Interfaz Intuitiva:** Sistema de menús y ventanas emergentes para consultas rápidas.

## 🛠️ Tecnologías Usadas

* **Lenguaje:** Python 3.12
* **Interfaz Gráfica:** Tkinter (CustomTkinter en el futuro)
* **Motor Astronómico:** Ephem
* **Arquitectura:** MVC (Modelo-Vista-Controlador)

## 🔧 Instalación y Uso

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/EmiDelgadoV/Lunaria.git
    ```
2.  Instala las dependencias:
    ```bash
    pip install ephem
    ```
3.  Ejecuta la aplicación:
    ```bash
    python main.py
    ```
